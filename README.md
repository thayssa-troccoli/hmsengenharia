HMS Engenharia — Website Institucional

Website responsivo desenvolvido para construtora HMS Engenharia e Construções.

Este repositório contém o código-fonte completo do site institucional da HMS Engenharia, desenvolvido com HTML, CSS, Bootstrap e JavaScript.
O foco do projeto foi entregar um site responsivo, leve e fácil de navegar, apresentando os serviços, portfólio e informações da empresa.

⚙️ Tecnologias Utilizadas

HTML5 — estrutura principal da aplicação
- CSS3 / Bootstrap 5 — responsividade e componentes visuais
- JavaScript Vanilla — interações básicas e inicialização de plugins
- AOS (Animate On Scroll) — animações de entrada
- Swiper.js — sliders e carrosséis
- Glightbox — galeria de imagens com lightbox
- Boxicons + Bootstrap Icons — ícones vetoriais

Outros Recursos
- Google Fonts (Open Sans, Raleway, Poppins)
- Google Maps Embed para localização
- Formulário com validação JS
